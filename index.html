Política de Privacidade

1. Dados que Recolhemos
Recolhemos apenas os dados necessários para prestar o serviço: nome, email, e dados técnicos dos sites WordPress que monitorizas (versão WP, plugins, métricas de performance e segurança).

2. Como Usamos os Dados
Os dados são usados exclusivamente para: prestar o serviço de monitorização, enviar alertas e notificações que solicitaste, e melhorar o produto. Nunca vendemos os teus dados a terceiros.

3. Armazenamento e Segurança
Os dados são armazenados em servidores seguros com encriptação em trânsito (HTTPS/TLS) e em repouso. As passwords são encriptadas com bcrypt (factor 12). Nunca armazenamos passwords em texto simples.

4. Cookies e Sessões
Usamos localStorage para guardar o token de autenticação da sessão. Não usamos cookies de rastreamento ou publicidade.

5. Partilha de Dados
Não partilhamos os teus dados com terceiros, excepto quando obrigados por lei. Os dados técnicos dos teus sites são exclusivamente teus.

6. Os Teus Direitos (RGPD)
Tens direito a aceder, rectificar, exportar e eliminar os teus dados a qualquer momento. Para exerceres estes direitos, contacta-nos via formulário de contacto. Respondemos em 30 dias.

7. Retenção de Dados
Os dados são mantidos enquanto a tua conta estiver activa. Após cancelamento, os dados são eliminados em 90 dias, salvo obrigação legal.

8. Contacto
Para questões de privacidade, contacta-nos através do formulário disponível na landing page. Comprometemo-nos a responder em 48 horas úteis.
